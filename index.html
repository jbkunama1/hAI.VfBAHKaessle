VfB Grötzingen – AH Bierkässle

hAI.VfBAHKaessle

Web-App & Telegram-Bot für das AH-Bierkässle: Biere nach dem Training erfassen, Monatsstände einsehen und alles sauber per SQLite verwalten.

Status & Technik

Projektüberblick

  • Flask-Web-App mit Login & Biereinträgen
  • SQLite-Backend im `instance/`-Ordner
  • Monatsübersicht pro Spieler + Gesamtsumme
  • Telegram-Bot zum Buchen & Status abfragen
  • Docker-/Portainer-Setup auf Port 1904

Tech-Stack

  • Backend: Python, Flask, Gunicorn
  • Datenbank: SQLite
  • Bot: python-telegram-bot
  • UI: Bootstrap 5, responsive

Telegram-Flow (Kurzfassung)

  1. Im Web einen Benutzer registrieren.
  2. Bot im Telegram starten, /start.
  3. Account verknüpfen: /link deinusername.
  4. Biere buchen: /bier oder /bier 3.
  5. Status abrufen: /status.

Deployment

Für den Betrieb gibt es ein Docker-/Compose-Setup mit zwei Containern (Web & Bot) und einem gemeinsamen Datenbank-Volume.

Details zu Installation, Docker-Stack und Telegram-Konfiguration findest du in der README im Repository.
